# 2.Краткое описание проекта
Автоматизированная система "Отдел кадров" предназначена для оптимизации выполнения задач управления данными (создание, редактирование, удаление, формирование выходных документов) в кадровой службе предприятия Заказчика. Программа автоматизирует учет состава семьи сотрудников, что критически важно для социальной поддержки внутри организации. 
# 3. Используемый стек технологий
**Язык программирования:** С# (.Net Framework 4.7.2).
**Клиентская часть:** Windows Forms. 
**Серверная часть / СУБД:** Microsoft SQL Server (LocalDB). 
 **Доступ данным:** Entity Framework Core
 **Среда разработки:** Visual Studio 2022
 **Контроль версий:** Git+GitHub.
 # 4. Описание основного функционала
 Клиент-серверное приложение имеет возможность работы с базой данных по описанной предметной области:
 **Отдел** (номер отдела, название отдела);
 **Сотрудник** (табельный номер сотрудника, ФИО сотрудника, стаж работы, пол);
 **Дети** (номер свидетельства о рождении, год рождения, имя, пол). 
 Созданное приложение будет выводить указанных в задании данные, производить создание (добавление) данных, их корректировку и удаление, а также выводить выходные документы. По запросам пользователей система выдает:
информацию о детях сотрудников определенного отдела, отсортированную по ФИО сотрудников; сгруппированный список всей базы данных автоматизированной системы. Группировка выполнена по отделам предприятия Заказчика, сортировка осуществляется по названию отделов и ФИО сотрудников. 
 # 5. Инструкция по запуску проекта
 ## Требования
 Windows 10/11; .Net Framework 4.7.2; Microsoft SQL Server 2022 LocalDB; Visual Studio 2022 (Community)
## Развертывание базы данных
Откройте SQL Server Management Studio. Выполните скрипт 01_Schema.sql для создания базы данных и таблиц.
Выполните скрипты 02_Views.sql, 03_OtdelProcedures.sql, 04_SotrudnikProcedures.sql, 05_DetyProcedures.sql для создания представлений и хранимых процедур.
## Настройка подключения
В файле Client/App.config в строке подключения укажите актуальное имя сервера и базы данных. 
## Запуск
Откройте файл GorbunovAEOKadrov.sln в Visual Studio. Убедитесь, что проект Client установлен как StartUp Project. Нажмите F5 для запуска клиентского приложения.
